# ReShaped

ReShaped is an open-source design studio building better interfaces for public-facing services.

## What we build

Most design systems are dictionaries: here is a button, here is a card. ReShaped is a grammar. It defines not just what the pieces are, but how they relate and compose.

That grammar has three parts:

- **[Objects](https://github.com/reshaped-studio/objects)** — a public library of UX object definitions using OOUX methodology.
- **[Shapes](https://github.com/reshaped-studio/shapes)** — a design system built around composition rules, not components.
- **Redesigns** — open-source rebuilds of services that are harder to use than they should be.

Each redesign draws from Objects and Shapes and contributes back to both.

## How we work

We design inclusively. If it works for someone with low literacy, low bandwidth, or low trust in institutions, it works for everyone.

We design iteratively. Everything ships at good enough to learn from.

We design intuitively. No feature earns its place without a user need.
